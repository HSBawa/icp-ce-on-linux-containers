Install Istio:<br>
```
  * cd <git clone dir for icp-ce-on-linux-containers>
  * sudo ./icp-setup/k8s/istio/istio-setup.sh install
```
<br>

Uninstall Istio:<br>
```
   cd <git clone dir for icp-ce-on-linux-containers>
   sudo ./icp-setup/k8s/istio/istio-setup.sh uninstall
```
